# .partials

> Reusable markdown snippets shared across multiple pages.

This directory is useful when the same warning, notice, or example needs to appear in more than one document.

## Example

The docs in this repository reuse a shared warning block with `automd:file`:

```md
<!-- automd:file src="../../.partials/warn.md" -->

> [!IMPORTANT]
> NuDocs is currently intended for NuJS docs only and is not fully customizable yet. <br>
> Contributions are more than welcome but please consider that this project is not ready yet to be used. <br>
> We don't guarantee stability yet and it is expected that it doesn't work time to time.

<!-- /automd -->
```

## When To Use It

- Shared warnings or notices
- Repeated setup instructions
- Reused markdown fragments that should stay in sync

If you only need content once, keep it inline in the page instead of moving it into `.partials/`.
