# package.json

> The docs project's scripts and dependencies.

The scaffold generated by `npm create nudocs` keeps this file minimal:

```json
{
  "name": "docs",
  "private": true,
  "scripts": {
    "dev": "nudocs dev",
    "build": "nudocs build"
  },
  "devDependencies": {
    "nudocs": "latest"
  }
}
```

## Typical Usage

- `pnpm dev` starts the local docs app.
- `pnpm build` generates `docs/.output/`.

You can add your own dependencies here when custom pages or components need them.
