# .output

> Generated production build output.

Running `pnpm build` writes the production output to `docs/.output/`.

## What Is Inside

- `.output/` is the full Nitro build output directory.
- `.output/public/` is the generated static site most hosting providers should publish.

## Important Notes

- Do not edit files in `.output/` manually.
- Delete and rebuild it if you want a clean production output.
- `.output/` is separate from `.docs/`, which contains your source configuration and assets.
