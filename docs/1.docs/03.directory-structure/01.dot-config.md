# .config

> Configuration files for the docs project live here.

NuDocs loads docs configuration from `.config/` in the project root.

## Common Files

- `.config/docs.yaml` is the main site configuration loaded by the CLI.
- `.config/automd.ts` is optional and registers custom `automd` generators.

## When To Use It

Use `.config/` for project-level settings and generator definitions. Keep static assets and theme resources in `.docs/`, and keep markdown content in your numbered content folders.
