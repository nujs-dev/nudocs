# .docs

> Static assets and optional app-level extensions live here.

This directory is mounted as part of the docs app and is where NuDocs-specific runtime assets and theme extensions live.

## Common Files

- `.docs/public/` contains static assets copied into the generated site root.
- `.docs/icons/` is optional and registers a custom icon collection with the `custom:` prefix.
- `.docs/pages/` can extend or replace app routes.
- `.docs/components/` can override bundled Vue components.

## When To Use It

Use `.docs/` for files that support the docs app itself, not for configuration files or markdown content. Put config in `.config/` and keep regular documentation pages in your numbered content folders.
