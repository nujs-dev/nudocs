# node_modules

> Installed dependencies and generated runtime files.

Like any Node-based project, this directory contains installed packages. In addition, NuDocs creates `node_modules/.nudocs/` as a generated working directory for its runtime app, build files, and caches.

## Important Notes

- `node_modules/.nudocs/` is an internal working directory, not your publish output.
- The final static site still lives in `docs/.output/`.
- You normally do not edit anything under `node_modules/` by hand.
